> Notion에 붙여넣어 쓰시면 됩니다. 급하실 땐 [한 장 요약](관장님_한장요약.md)만 보셔도 충분합니다.

---

## 0. 이 앱이 하는 일

**회원**은 폰으로 수업을 예약하고, **관장님**은 명단을 보고 출석을 체크합니다.

- 카톡으로 예약 받고 일일이 답장하지 않아도 됩니다
- 정원이 넘게 받아 곤란해질 일이 없습니다 (자동 대기 처리)
- 회원권이 끝나가는 분이 자동으로 표시됩니다
- 노쇼가 기록으로 남습니다

**비용은 들지 않습니다.** 무료 범위 안에서 돌아갑니다.

---

## 1. 시작하기

### 앱 열기
주소를 폰 브라우저에서 열고, **홈 화면에 추가**하면 앱처럼 아이콘이 생깁니다.

- **아이폰**: 공유 버튼 → "홈 화면에 추가"
- **안드로이드**: 메뉴(⋮) → "홈 화면에 추가"

### 로그인
- **관장님(관리자)**: 앱 첫 화면 아래 **[관리자 로그인]** → 이메일·비밀번호
- **회원**: 이름 + 전화번호 뒤 4자리

---

## 2. 매일 쓰는 화면 — 타임슬롯

관리자로 로그인하면 처음 보이는 화면입니다.

### 명단 확인
달력에서 날짜를 누르면 그날 수업이 아래에 나옵니다.
수업마다 **예약자 이름·전화번호**가 보이고, 정원이 차면 **대기자**가 순서대로 표시됩니다.

- 🏷️ **신규** 배지 = 등록한 지 7일 이내 회원
- **점(●)이 찍힌 날짜** = 예약이 있는 날
- **흐린 날짜** = 수업이 없는 날
- 지난 날짜도 눌러서 볼 수 있습니다 (출석 기록 확인용)

### 출석 체크
이름 옆 **[출석]** / **[노쇼]** 를 누르면 됩니다. 잘못 눌렀으면 다시 눌러서 해제됩니다.

> 회원이 수업 당일 직접 **[출석]** 을 누를 수도 있습니다. 그러면 관장님은 안 하셔도 됩니다.
> 아무도 안 누른 채로 수업이 지나면 회원 화면에 **미출석**으로 표시됩니다.

### 수업 취소 (휴강)
수업 오른쪽 **[취소]** → 세 가지 중 선택합니다.

| 선택 | 언제 쓰나 |
|---|---|
| **이 수업만** | 이번 한 번만 쉬는 경우 |
| **이 날 전체** | 그날 모든 수업을 안 하는 경우 |
| **앞으로 계속** | 그 시간 수업을 아예 없애는 경우 (시간표에서도 빠집니다) |

취소하면 그 수업의 예약도 함께 취소됩니다.
**실수하셔도 [복구] 버튼으로 되돌릴 수 있습니다.**

### 자율운동 시간
수업 대신 자유롭게 운동하는 시간으로 바꾸려면 **[자율운동으로]** 를 누릅니다.
자율운동은 **주간 횟수에서 차감되지 않습니다.**

### 슬롯 채우기
**[N월 슬롯 채우기]** 를 누르면 그 달의 수업이 시간표대로 만들어집니다.
평소엔 자동으로 채워지니 안 누르셔도 됩니다. 시간표를 바꾼 직후에만 쓰시면 됩니다.

## 3. 회원 관리

### 새 회원 등록
**회원** → **[신규 등록]**

1. 이름, 전화번호 입력
2. 이용권 입력 (시작일·종료일·주간 횟수)
   - 주간 횟수 = 주 2회권이면 2, 3회권이면 3
3. 규정을 보여드리고 **손가락으로 서명** 받기

> 이용권이나 서명은 나중에 해도 됩니다. 비워두고 등록하셔도 돼요.

### 종이 명부 옮기기
**[일괄 등록]** 을 누르고 아래 형식으로 붙여넣으면 한 번에 등록됩니다.

```
홍길동,010-1234-5678,2026-07-01,2026-08-01,3,3개월 등록
김철수,010-2222-3333
```

`이름,전화번호,시작일,종료일,주간횟수,메모` 순서입니다. 뒷부분은 생략해도 됩니다.
엑셀에서 복사해 붙여넣어도 됩니다. 잘못된 줄만 따로 알려드립니다.

### 회원권 연장
**[이용권]** → **[추가]** 에 새 기간을 넣습니다.

> 기존 것을 고치지 않고 **새로 추가**하는 게 맞습니다. 그래야 "지난달엔 어땠지?"를 볼 수 있습니다.
> **[수정]** 은 날짜를 잘못 입력했을 때만 쓰세요.

### 고정 수업 (정기 예약)
"화·목 19시 고정" 같은 분은 **[고정]** 에서 요일과 시간을 지정해두면
수업이 만들어질 때 **자동으로 예약**됩니다.

회원은 일정이 생기면 그 날만 취소하고 다른 날로 옮기면 됩니다.

### 그 밖에
- **[수정]**: 이름·전화번호 변경, 회원 비활성화
- **[서명받기]**: 서명을 아직 안 받은 회원에게 나중에 받기
- **[서명]**: 받아둔 서명 확인

## 4. 주간 시간표

**반복되는 수업 시간표**를 정합니다. 여기서 정한 대로 매주 수업이 만들어집니다.

### 등록하기
1. 코치 이름, 정원 입력
2. 요일 선택 (여러 개 가능)
3. 시간 선택 (여러 개 가능)
4. **[추가/변경]**

예) 월·수·금 + 18시·20시 → 6개 수업이 한 번에 만들어집니다.

### 시간표 바꾸기
**[이 시간표로 맞추기]** 를 쓰면 선택한 대로 **통째로 맞춰집니다.**

예) 지금 월수금 18·19·20시인데 19시를 없애고 싶다면
→ 월·수·금 선택 + 18시·20시만 선택 → **[이 시간표로 맞추기]**
→ 19시가 사라집니다. 몇 개가 없어지는지 미리 알려드립니다.

> ⚠️ **이미 만들어진 수업은 바뀌지 않습니다.** 다음에 수업이 만들어질 때부터 적용됩니다.
> 이미 만들어진 걸 없애려면 타임슬롯 화면에서 취소하세요.

### 휴관일
명절이나 쉬는 날을 등록해두면 그날은 수업이 만들어지지 않습니다.
지난 휴관일도 기록으로 남습니다.

## 5. 대시보드

**기간별 통계**를 봅니다. 기본은 이번 달입니다.

- 총 예약 수, 출석률, 취소 건수
- **요일별·시간대별 예약** → 어느 시간이 인기인지
- **최다 출석 회원**
- **회원권 만료 임박** 명단 → 재등록 안내하기 좋습니다

## 6. 설정

| 항목 | 설명 |
|---|---|
| **취소 패널티** | 수업 시작 몇 시간 전부터 취소를 막을지 |
| **노쇼 차감** | 노쇼도 주간 횟수에서 뺄지 |
| **체육관 규정** | 회원 등록할 때 보여주고 서명받는 내용 |
| **공지사항** | 회원 화면 맨 위에 뜨는 알림 (지난 공지도 보관됩니다) |
| **운영시간** | 요일별 여는 시간·닫는 시간 |
| **관리자** | 관리자 계정 추가·삭제 |

## 7. 회원분들께 안내할 내용

> 예약은 앱에서 하시면 됩니다.
> **주소:** ____________________
>
> - **로그인**: 이름 + 전화번호 뒤 4자리
> - 날짜를 누르고 원하는 시간에 **[예약하기]**
> - 정원이 차 있으면 **[대기 신청]** — 누가 취소하면 자동으로 예약됩니다
> - 수업 당일 오셔서 **[출석]** 을 눌러주세요
> - 취소는 수업 시작 **○시간 전까지** 가능합니다

## 8. 자주 묻는 것

**Q. 잘못 눌렀어요**
대부분 되돌릴 수 있습니다. 수업 취소는 **[복구]**, 회원 비활성화도 다시 활성화됩니다.

**Q. 회원이 예약이 안 된다고 해요**
- 회원권이 만료됐거나 (회원 목록에서 확인)
- 그 주 횟수를 다 썼거나 (주 2회권인데 이미 2번)
- 같은 날 이미 다른 수업을 예약했거나 (하루 1회)
- 정원이 차서 대기로 들어갔거나

회원 화면에 이유가 표시됩니다.

**Q. 회원이 폰을 바꿨어요**
아무 문제 없습니다. 이름과 전화번호 뒤 4자리로 다시 로그인하면 됩니다.

**Q. 전화번호가 바뀌었어요**
회원 → **[수정]** 에서 바꿔주세요. 그다음부터는 새 번호 뒤 4자리로 로그인합니다.
